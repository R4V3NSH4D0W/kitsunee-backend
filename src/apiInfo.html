API Documentation

Explore and test the available APIs for GogoAnime and ZoroAnime services.

GogoAnime Routes

  • GET /api/gogoanime/search?q=query

    Search anime with a query string.

  • GET /api/gogoanime/topairing?page=1

    Fetch top airing anime by page.

  • GET /api/gogoanime/details?id=anime-id

    Fetch details of a specific anime.

ZoroAnime Routes

  • GET /api/zoroanime/search?q=query

    Search anime with a query string.

  • GET /api/zoroanime/popularanime

    Fetch popular anime.

  • GET /api/zoroanime/recentepisodes

    Fetch recently uploaded episodes.
